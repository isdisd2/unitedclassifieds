# Flight search demo app

## Instalation:

cd <instalation_dir>\unitedclassifieds-api-node-hi\
npm i\
npm start\
cd <instalation_dir>\unitedclassifieds-api-node-server\
npm i\
npm start

## Usage:

open in browser site
http://localhost:8080/

## Limitations:
max. 20 flights items in list
